Shrubbery is an online Python environment for teaching programming. It aims to solve the problem of having to setup any sort of environment on shared computers for teaching programming. Instead it provides a text editor and Python interpreter in the browser.

It's based upon Skulpt for the Python interpreter and CodeMirror for the text editor. It also has well documented examples to show you what you can do.

It was thrown together in a few days for an introductory Python tutorial I was teaching. It's licensed under the MIT license and it's on GitHub (probably with multiple issues).
